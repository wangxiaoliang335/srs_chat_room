# 翻译服务完整接口文档

## 一、语音翻译服务（回调服务器）

### 服务器地址
```
http://<服务器IP>:8085
```

---

### 1. 申请翻译

**接口：**
```http
POST /api/v1/translation/request
Content-Type: application/json
```

**请求参数：**
| 参数 | 必填 | 说明 |
|------|------|------|
| room_id | 是 | 房间ID |
| source_user | 是 | 说话人用户ID |
| target_user | 是 | 申请翻译的用户ID |
| to_lang | 否 | 目标语言代码，默认 "zh" |

**请求示例：**
```json
{
    "room_id": "room1",
    "source_user": "A",
    "target_user": "B",
    "to_lang": "zh"
}
```

**响应示例：**
```json
{
    "code": 0,
    "message": "success",
    "data": {
        "request_id": "f0c8d7ae-6c1f-4524-87a1-3f803714da97",
        "stream_url": "rtmp://localhost:1935/live/room1_A_to_zh"
    }
}
```

---

### 2. 取消翻译

**接口：**
```http
POST /api/v1/translation/cancel
Content-Type: application/json
```

**请求参数（二选一）：**
```json
// 方式1：通过request_id
{
    "request_id": "f0c8d7ae-6c1f-4524-87a1-3f803714da97"
}

// 方式2：通过参数
{
    "room_id": "room1",
    "source_user": "A",
    "target_user": "B",
    "to_lang": "zh"
}
```

**响应示例：**
```json
{
    "code": 0,
    "message": "success"
}
```

| type | 说明 |

## 三、客户端集成示例

### 1. 连接语音翻译服务（申请翻译）

```javascript
// 申请翻译
async function requestTranslation(roomId, sourceUser, targetUser, toLang) {
    const response = await fetch('http://localhost:8085/api/v1/translation/request', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            room_id: roomId,
            source_user: sourceUser,
            target_user: targetUser,
            to_lang: toLang
        })
    });
    
    const result = await response.json();
    if (result.code === 0) {
        console.log('翻译请求创建成功');
        console.log('翻译流地址:', result.data.stream_url);
        return result.data;
    } else {
        console.error('申请翻译失败:', result.message);
    }
}

// 使用示例：用户B申请翻译用户A的语音为中文
requestTranslation('room1', 'A', 'B', 'zh');
```

### 2. 连接翻译文本推送服务（接收文本）

```javascript
class TranslationTextClient {
    constructor(userId) {
        this.userId = userId;
        this.ws = null;
        this.reconnectInterval = 5000;
        this.isConnected = false;
    }
    
    connect() {
        this.ws = new WebSocket(`ws://localhost:8086/ws/translation/${this.userId}`);
        
        this.ws.onopen = () => {
            console.log('翻译文本服务已连接');
            this.isConnected = true;
        };
        
        this.ws.onmessage = (event) => {
            const message = JSON.parse(event.data);
            this.handleMessage(message);
        };
        
        this.ws.onclose = () => {
            console.log('连接断开，尝试重新连接...');
            this.isConnected = false;
            setTimeout(() => this.connect(), this.reconnectInterval);
        };
        
        this.ws.onerror = (error) => {
            console.error('WebSocket错误:', error);
        };
    }
    
    handleMessage(message) {
        switch (message.type) {
            case 'connected':
                console.log('已连接:', message.message);
                break;
                
            case 'translation_text':
                this.onTranslationText(message.data);
                break;
                
            case 'cached_texts':
                console.log('收到缓存翻译:', message.data.texts);
                message.data.texts.forEach(text => this.onTranslationText(text));
                break;
                
            case 'error':
                console.error('错误:', message.message);
                break;
        }
    }
    
    onTranslationText(data) {
        console.log('--- 收到翻译 ---');
        console.log('说话人:', data.source_user);
        console.log('原文:', data.original_text);
        console.log('译文:', data.translated_text);
        console.log('源语言:', data.source_lang, '->', '目标语言:', data.target_lang);
        
        // 更新UI
        this.displayTranslation(data);
    }
    
    displayTranslation(data) {
        // 创建翻译文本显示
        const container = document.getElementById('translation-container');
        const div = document.createElement('div');
        div.className = 'translation-item';
        div.innerHTML = `
            <div class="speaker">👤 ${data.source_user}</div>
            <div class="original">${data.original_text}</div>
            <div class="translated">${data.translated_text}</div>
        `;
        container.appendChild(div);
        
        // 滚动到底部
        container.scrollTop = container.scrollHeight;
    }
    
    disconnect() {
        if (this.ws) {
            this.ws.close();
            this.ws = null;
        }
    }
}

// 使用示例
const client = new TranslationTextClient('B');
client.connect();

// 离开页面时断开连接
window.addEventListener('beforeunload', () => {
    client.disconnect();
});
```

### 3. 完整使用流程

```javascript
// 1. 用户B申请翻译用户A的语音为中文
const result = await requestTranslation('room1', 'A', 'B', 'zh');

// 2. 用户B可以订阅翻译流地址进行收听
// 翻译流地址: rtmp://localhost:1935/live/room1_A_to_zh

// 3. 同时连接文本推送服务，实时接收翻译文本
const textClient = new TranslationTextClient('B');
textClient.connect();

// 4. 当不需要翻译时，取消翻译
await fetch('http://localhost:8085/api/v1/translation/cancel', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        room_id: 'room1',
        source_user: 'A',
        target_user: 'B',
        to_lang: 'zh'
    })
});

// 5. 断开文本推送连接
textClient.disconnect();
```

## 七、数据库

翻译文本会持久化保存到 SQLite 数据库：

- **数据库路径：** `/tmp/translation_records.db`
- **可通过环境变量修改：** `TEXT_DB_PATH=/path/to/db`

**表结构：**
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键 |
| request_id | String | 翻译请求ID |
| room_id | String | 房间ID |
| source_user | String | 说话人用户ID |
| target_user | String | 目标用户ID |
| original_text | Text | 原文 |
| translated_text | Text | 译文 |
| source_lang | String | 源语言 |
| target_lang | String | 目标语言 |
| timestamp | Float | 时间戳 |
| created_at | DateTime | 创建时间 |

## 十、注意事项

1. **两个服务都需要启动**
   - 回调服务器（端口8085）：处理翻译请求、管理翻译服务
   - 文本推送服务（端口8086）：通过WebSocket推送翻译文本

2. **连接管理**
   - 用户离开房间时断开WebSocket
   - 实现心跳保活
   - 断线自动重连

3. **性能考虑**
   - 每个用户一个WebSocket连接
   - 服务器维护连接映射表

4. **消息格式**
   - 所有消息都是JSON格式
   - 包含时间戳用于排序
