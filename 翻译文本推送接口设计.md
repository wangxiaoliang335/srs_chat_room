# 翻译文本推送接口设计

## 方案：WebSocket 实时推送

### 1. WebSocket 连接

客户端连接翻译文本推送服务：

```http
ws://<服务器IP>:8086/ws/translation/{user_id}
```

**示例：**
```
ws://localhost:8086/ws/translation/B
```

### 2. 连接成功

服务器推送连接成功消息：

```json
{
    "type": "connected",
    "message": "Translation text service connected",
    "user_id": "B"
}
```

### 3. 收到翻译文本

当有翻译文本时，服务器实时推送：

```json
{
    "type": "translation_text",
    "data": {
        "request_id": "xxx",
        "room_id": "room1",
        "source_user": "A",
        "original_text": "Hello world",
        "translated_text": "你好世界",
        "source_lang": "en",
        "target_lang": "zh",
        "timestamp": 1709999999.123
    }
}
```

### 4. 推送类型汇总

| type | 说明 |
|------|------|
| connected | 连接成功 |
| translation_text | 翻译文本 |
| error | 错误信息 |

---

## HTTP 轮询方案（备选）

如果 WebSocket 不可用，也可以使用 HTTP 轮询：

```http
GET /api/v1/translation/text/{request_id}?last_timestamp=1709999000
```

**响应：**

```json
{
    "code": 0,
    "message": "success",
    "data": {
        "texts": [
            {
                "request_id": "xxx",
                "original_text": "Hello",
                "translated_text": "你好",
                "timestamp": 1709999999.123
            }
        ],
        "has_more": false
    }
}
```

## 客户端集成示例

### WebSocket 连接（推荐）

```javascript
// 连接翻译文本服务
const userId = 'B';
const ws = new WebSocket(`ws://localhost:8086/ws/translation/${userId}`);

ws.onopen = function() {
    console.log('Connected to translation text service');
};

ws.onmessage = function(event) {
    const message = JSON.parse(event.data);
    
    switch (message.type) {
        case 'connected':
            console.log('Connected:', message.message);
            break;
            
        case 'translation_text':
            console.log('原文:', message.data.original_text);
            console.log('译文:', message.data.translated_text);
            console.log('说话人:', message.data.source_user);
            // 在UI显示翻译文本
            displayTranslationText(message.data);
            break;
            
        case 'error':
            console.error('Error:', message.message);
            break;
    }
};

ws.onerror = function(error) {
    console.error('WebSocket error:', error);
};

ws.onclose = function() {
    console.log('Connection closed');
    // 可以在这里实现重连逻辑
};

// 显示翻译文本的UI示例
function displayTranslationText(data) {
    const chatContainer = document.getElementById('chat');
    const messageDiv = document.createElement('div');
    messageDiv.className = 'translation-message';
    messageDiv.innerHTML = `
        <div class="speaker">${data.source_user}</div>
        <div class="original-text">${data.original_text}</div>
        <div class="translated-text">${data.translated_text}</div>
    `;
    chatContainer.appendChild(messageDiv);
}
```

## 错误码

| code | 说明 |
|------|------|
| 0 | 成功 |
| 400 | 无效的user_id |
| 401 | 未授权（需要验证用户身份） |
| 500 | 服务器错误 |

---

## 注意事项

1. **连接管理**
   - 用户离开房间时断开 WebSocket
   - 实现心跳保活
   - 断线自动重连

2. **性能考虑**
   - 每个用户一个 WebSocket 连接
   - 服务器维护连接映射表

3. **消息格式**
   - 所有消息都是 JSON 格式
   - 包含时间戳用于排序
